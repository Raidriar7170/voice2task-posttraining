本阶段改了什么

  • 新增 train-split diagnostic config template，固定 prediction_split=train、overfit_diagnostic=true、generalization_claim=false。
  • prediction export 增加 prompt snapshot、sanitized raw decoded summary、generation trace 和 prediction metadata links。
  • Objective inspection 采用 fail-closed 表达：本地缺少训练依赖或真实 label evidence 时，mask/loss 字段为 null。
  • Reviewer Must Fix 后移除了 private prediction 的 fixture sidecar fallback，避免把 gold fixture 包装成 private adapter evidence。
  • metrics/report/manifest 补充 standalone 边界语境，避免 1.0000 fixture metrics 被误读为模型质量。
